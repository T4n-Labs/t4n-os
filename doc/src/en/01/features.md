
# Features

T4n OS provides:

## Lightweight

Optimized system with minimal background services.

---

Built on top of Void Linux and powered by `xbps`.

---

## Rolling Release

Always up-to-date without full reinstall.

---

## Developer Friendly

- Pre-configured development tools
- Clean terminal environment
- Minimal bloat

---

## Secure by Design

- Minimal attack surface
- Clean architecture
- Community-driven packages
