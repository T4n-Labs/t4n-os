# T4n OS

Selamat datang di dokumentasi resmi **T4n OS**.

T4n OS adalah distribusi Linux modern dan ringan yang dibangun di atas Void Linux,
dirancang untuk kinerja, kesederhanaan, dan efisiensi.

---

## Filosofi

> Membuat teknologi kompleks menjadi sederhana dan mudah diakses.

T4n OS berfokus pada:

- Minimalisme
- Kecepatan
- Stabilitas
- Lingkungan yang ramah pengembang
- Arsitektur sistem yang bersih

---

## Pengguna Sasaran

- Pengembang
- Penggemar Linux
- Administrator sistem
- Pembelajar keamanan
- Pengguna desktop tingkat lanjut

---

## Struktur Proyek

T4n OS dibangun dengan:

- Basis Void Linux
- Manajer paket xbps
- Peningkatan khusus
- VUR (Void User Repository)
