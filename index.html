Tentang T4n OS

T4n OS adalah Distribusi Linux yang berbasis Void Linux. Sistem ini dirancang untuk pengguna yang menginginkan kontrol penuh (Power User) dan pengguna umum untuk keperluan: Browsing, Office, Content Creator, Editor, Programmer dan Server.

Dengan integrasi VUR (Void User Repository), T4n OS menyediakan akses ke ribuan paket tambahan yang tidak tersedia di repository resmi Void Linux, memberikan fleksibilitas maksimal untuk pengguna.
